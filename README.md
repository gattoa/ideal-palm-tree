# ideal-palm-tree
to do list 

We are building a to-do list application.

Features:
- Add New To-dos
- Mark as Complete (and undo)
- Delete a To-do

Tech
- Vite
- VanillaJS
- No Backend for now



## Vision
I want it to feel grounded, but approachable. 

The goal is for the user to feel their progress is accumulating to something bigger. 

There will be plateus in the customer's journey and their progress. But they should always be able to see thier journey, where there going, and feel the wins as they come.

This is tool is for keeping them sane, grounded, motivated and fueled during those long, seemingly meaningless days between wins. This app let's them know that these days, these tasks, they are not wasted effort. They mean something, and they are accomplishments, even if they don't feel like them.

The brand and style should reflect this, and communicate that emotion visually, through composition of text, color, shape, space and image.
